#  Build Your Own Jarvis — MCU × AIMLA Workshop

A 3-hour hands-on event where you build a real AI assistant that 
**thinks** and **speaks** from scratch.

## ⏱ Schedule at a glance

| Checkpoint | Time | Topic |
|---|---|---|
| [0 — Setup](./checkpoint-0/) | 0–20 min | Install, clone, get your API key |
| [1 — LLM core](./checkpoint-1/) | 20–40 min | Talk to an LLM via API |
| [2 — Voice](./checkpoint-2/) | 60–80 min | Make Jarvis speak out loud |
| [3 — Vision](./checkpoint-3/) | 80–100 min | Give Jarvis eyes | (super maybe)
| [4 — Full Jarvis](./checkpoint-4/) | 120–180 min | Combine everything |
| [Bonus](./bonus/) | If you finish early | Personality, memory, webcam |

> **Tip:** Each folder has its own README with step-by-step instructions.  
> If you're ahead, jump to the next checkpoint or check out [bonus/](./bonus/).  
> Facilitators will check in every ~20 minutes.

## 🚀 Quick start

\```bash
git clone https://github.com/YOUR-ORG/jarvis-workshop
cd jarvis-workshop/checkpoint-0
pip install -r requirements.txt
\```



## 🧠 What you'll build

- An LLM that answers questions about any topic
- A voice that reads responses aloud  
- A single unified `jarvis_main.py` that combines all three
